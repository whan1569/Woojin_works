# 나라배터리 원격지원 진행 보고서

작성일: 2026-06-24

상태: 진행 중

## 1. 개요

본 보고서는 나라배터리 설비 원격지원 과정에서 발생한 원격 연결 불가 이슈와 이에 대한 확인 및 조치 내용을 정리한 것이다.

초기 목적은 원격 접속을 통해 설비 상태 및 기계 동작 원인을 확인하는 것이었으나, 원격 접속 단계에서 연결이 정상적으로 이루어지지 않아 우선 원격 연결 환경을 점검하였다.

## 2. 작업 목적

이번 원격지원의 목적은 다음과 같다.

1. 설비와 원격지원 PC 간 연결 상태 확인
2. 원격 접속 가능 여부 확인
3. 원격 접속 후 실제 설비 상태 및 기계 원인 확인
4. 필요 시 프로그램 설정 및 현장 상태 점검

## 3. 진행 내용

### 3.1 원격 연결 상태 확인

원격지원 시작 시 설비 측 네트워크 연결이 정상적으로 이루어지지 않는 문제가 발생하였다.

이에 따라 우선 다음 항목을 확인하였다.

* 현장 LAN 케이블 연결 상태
* 연결 대상 LAN 포트 위치
* 포트 번호 및 네트워크 설정 상태
* 설비 측 Ethernet 설정 상태
* Automation Studio 기준 원격 접속 관련 ETH 포트 설정 여부

확인 결과, 단순 네트워크 연결 문제뿐 아니라 설비 프로그램 측에서 원격 접속을 위한 ETH 포트가 개방되어 있지 않은 상태임을 확인하였다.

### 3.2 원격 접속 불가 원인 확인

해당 설비는 구형 프로그램 구성 기준으로 생성된 것으로 보이며, 기존에는 보안 처리를 목적으로 원격 연결에 필요한 포트를 기본적으로 차단해두는 방식이 적용되어 있었다.

이로 인해 원격지원 시점에서 바로 설비에 접속할 수 없었고, 현장에서 해당 포트를 열어주는 절차가 선행되어야 했다.

확인된 원인은 다음과 같다.

> 원격 접속에 필요한 ETH 포트가 프로그램 생성 단계에서 사전에 개방되어 있지 않아, 원격 연결 자체가 불가능한 상태였다.

### 3.3 조치 내용

원격 접속이 가능하도록 Automation Studio에서 해당 ETH 포트 관련 설정을 확인하고, 원격 연결에 필요한 포트를 개방하는 방향으로 설정을 수정하였다.

이후 해당 설정을 반영한 프로그램을 재생성하여, 다음 원격지원 시 사용할 수 있도록 준비 방향을 확정하였다.

조치 내용은 다음과 같다.

* AS 프로젝트 내 ETH 포트 설정 확인
* 원격 접속에 필요한 포트 개방 방향 확인
* 구형 프로그램의 원격 접속 차단 구조 확인
* 원격지원용 포트 설정을 반영한 프로그램 재생성 필요성 확인
* 다음 원격지원 시 적용할 프로그램 준비 방향 확정

## 4. 작업 중단 사유

원격 연결 문제 확인 및 포트 개방 방향까지는 확인하였으나, 현장 설비를 계속 운전해야 하는 상황이 발생하여 작업을 일시 중단하였다.

이에 따라 설비는 기존 상태로 원복하였으며, 실제 기계 원인 분석은 다음 원격지원 시 재진행할 예정이다.

이번 작업에서는 원격 접속을 위한 사전 조건 확인 및 해결 방향 확정까지 진행하였다.

## 5. 현재까지의 결과

이번 작업을 통해 다음 내용을 확인하였다.

1. 기존 상태에서는 원격 접속이 불가능한 구조였음
2. 원인은 설비 프로그램 내 원격 접속용 ETH 포트 미개방 상태였음
3. 구형 프로그램에서는 보안 목적의 포트 차단 방식이 적용되어 있었음
4. 최신 기종에서는 원격지원용 전용 포트를 사전에 구성하여 출하하는 방식이 적용되고 있음
5. 해당 설비도 원격지원용 포트를 사전 개방한 프로그램을 재생성하여 적용할 필요가 있음
6. 다음 원격지원에서는 사전 준비된 프로그램을 기준으로 원격 접속부터 다시 진행해야 함

## 6. 현장 지원 과정에서 확인된 사항

원격지원 과정에서 현장 측 네트워크 연결 준비가 충분하지 않아 연결 확인에 추가 시간이 소요되었다.

특히 LAN 케이블 연결 위치, LAN 포트 구분, 포트 번호 설정 등 기본 네트워크 연결 절차를 원격으로 안내하며 진행해야 했다.

이에 따라 향후 원격지원 전에는 현장 담당자에게 다음 항목을 사전에 전달할 필요가 있다.

* 연결해야 할 LAN 포트 위치
* 사용할 LAN 케이블 위치
* 포트 번호 및 네트워크 설정 방법
* 원격지원 전 설비 준비 상태
* 작업 중 대기 및 응답 절차
* 원복 필요 시 절차

## 7. 향후 진행 계획

다음 원격지원 전까지 원격 접속용 ETH 포트를 개방한 프로그램을 사전에 준비한다.

다음 작업은 아래 순서로 진행할 예정이다.

2. 기존 프로그램 백업 및 원복 절차 확인
3. 현장 담당자에게 LAN 연결 위치 및 설정 절차 사전 안내
4. 원격 접속 가능 여부 재확인
5. 설비 상태값 및 I/O 상태 확인
6. 실제 기계 동작 원인 분석

## 8. 결론

이번 원격지원에서는 실제 기계 원인 분석 단계로 진입하기 전, 원격 접속 자체가 불가능한 문제가 먼저 확인되었다.

원인은 구형 프로그램 구성에서 원격 접속용 ETH 포트가 보안 목적상 개방되어 있지 않았기 때문이다.

이에 따라 Automation Studio 기준으로 해당 ETH 포트를 개방한 프로그램을 사전에 재생성하여 적용해야 한다.

이번 작업은 현장 설비 운전 일정으로 인해 중단 및 원복되었으며, 다음 원격지원에서는 사전 준비된 프로그램을 기준으로 원격 접속을 재진행한 후 실제 설비 원인을 확인할 예정이다.
